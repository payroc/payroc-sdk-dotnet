# Dotnet SDK Functional Tests (Internal)

## Environment Setup

The following environment variables must be set:

`PAYROC_API_KEY_GENERIC`
(this will be used for reporting and notifications)

`PAYROC_API_KEY_PAYMENTS`
(this will be used for the payment endpoints)

`TERMINAL_ID_NO_AVS`
(this will be needed for payment tests)

`TERMINAL_ID_AVS`
(this will be needed for payment tests)
